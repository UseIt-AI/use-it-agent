## Quick Start
The intended first-run experience:
```bash
opencreativework onboard
opencreativework agent --message "Open my design file and prepare export variants"
```
Example tasks we want OpenCreativeWork to support:
- "Open this Blender scene, adjust the camera, render three product shots, and export them."
- "Take this Figma frame, clean up spacing, create mobile variants, and prepare a handoff package."
- "Open this video project, find the rough cuts, add captions, balance audio, and export a review file."
- "Load this CAD file, inspect the part, generate screenshots, and prepare notes for manufacturing."
## Contributing
OpenCreativeWork is for people who believe AI should work with the tools creators already use.
We welcome contributions across:
- Creative software adapters
- Computer control and UI automation
- Agent runtime design
- Model evaluation for creative tasks
- Workflow examples and benchmarks
- Documentation, demos, and tutorials
If you are a designer, video editor, 3D artist, musician, engineer, researcher, or tool builder, your workflows matter here.
To contribute:
1. Open an issue describing the workflow, adapter, bug, or idea.
2. Keep pull requests focused and easy to review.
3. Include examples, screenshots, logs, or recordings when they help explain behavior.
4. Prefer real creative workflows over toy demos.
